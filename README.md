# 📱 ShadowFox Android Internship Projects

This repository contains the Android applications developed as part of the **ShadowFox Android Internship Program (2026)**.

The focus of this internship was not just building apps, but applying **engineering principles, problem-solving, and incremental innovations** to improve real-world applications.

---

# 🚀 Projects Overview

## 1. 🔐 Login Application

A secure login system with user authentication and input validation.

### Features:

* Email & Password validation
* Error handling for empty fields
* Firebase Authentication integration (real user login)
* Smooth navigation using Intents
* Secure handling of sensitive data

---

## 2. 🌦️ Weather Application (API Integration)

An Android app that fetches real-time weather data using REST APIs.

### Features:

* Weather data fetched using API (Retrofit)
* Automatic location detection using GPS
* Displays temperature, weather condition, etc.
* Handles no internet scenarios gracefully
* Background threading (no UI blocking)

---

## 3. 📘 Facebook Integration App

Integration of Facebook SDK for authentication and user data access.

### Features:

* Facebook Login (OAuth 2.0)
* Fetch user profile data (Name, Email, Profile Picture)
* Session management (auto login if already authenticated)
* Permission handling and error cases

---

# 🚀 Incremental Innovations (Key Highlights)

As per internship guidelines, the following improvements were implemented:

* 🔐 Replaced dummy login with **Firebase Authentication**
* 🌍 Implemented **GPS-based automatic location detection** in Weather App
* 📘 Extended Facebook Integration to fetch **user profile details (email & image)**

# ⚙️ How to Run the Project

1. Clone this repository
2. Open in Android Studio
3. Add required API keys:

   * Weather API Key
   * Firebase configuration (google-services.json)
   * Facebook App ID
4. Sync Gradle
5. Run on emulator or physical device
